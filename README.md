# Hiring-Process-Analytics# Hiring Process Analytics Dashboard 📊

An industry-grade Excel dashboard designed to analyze and optimize recruitment workflows. This project transforms raw hiring data into actionable insights regarding candidate demographics, salary distributions, and departmental performance.

---

## 🖼️ Dashboard Preview
![Hiring Process Dashboard](dashboard_screenshot.png)
*Full-screen interactive dashboard featuring custom slicers, sorted data visualizations, and optimized KPI tracking.*

---

## 🚀 Key Features & Functionality
* **KPI Tracking:** Real-time monitoring of Total Candidates, Hire Rate, Average Salary, and Median Salary.
* **Recruitment Funnel:** Analysis of 'Hired' vs. 'Rejected' status across various departments and post names.
* **Salary Benchmarking:** Horizontal bar charts for average salary by department and salary bracket distribution.
* **Dynamic Filtering:** Multi-select slicers for Department, Post Name, Gender, and Status for granular data exploration.
* **UX Optimized:** Custom UI design with hidden gridlines, headers, and automated zoom for a "software-app" feel.

---

## 🛠️ Tech Stack & Skills Applied
* **Tool:** Microsoft Excel (Advanced)
* **Data Modeling:** Star Schema design and PivotTable architecture.
* **Data Visualization:** Custom chart formatting, sorting logic, and UX/UI design principles.
* **VBA (Optional):** Automated interface cleanup and "Reset All Filters" functionality.
* **Functions:** `VLOOKUP`, `INDEX-MATCH`, `IFERROR`, and complex DAX-style measures in PivotTables.
